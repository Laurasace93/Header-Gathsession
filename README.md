# Header Gathsession 🚀

Este proyecto es una implementación de una **cabecera (header)** diseñada para ser moderna, limpia y funcional. Ideal como base para proyectos web que requieran una navegación clara y atractiva.

## 🛠️ Tecnologías utilizadas

* **HTML5**: Estructura semántica.
* **CSS3**: Estilos modernos, incluyendo Flexbox para el posicionamiento y diseño.
* **Google Fonts**: Integración de tipografía personalizada para un acabado profesional.

## 📋 Características

* **Código Limpio**: Estructura de archivos sencilla y fácil de mantener.

## 🚀 Instalación y Uso

Si quieres probar este componente localmente, sigue estos pasos:

1.  **Clona el repositorio:**
    ```bash
    git clone [https://github.com/Laurasace93/Header-Gathsession.git](https://github.com/Laurasace93/Header-Gathsession.git)
    ```
2.  **Navega a la carpeta del proyecto:**
    ```bash
    cd Header-Gathsession
    ```
3.  **Abre el archivo `index.html`** en tu navegador preferido.
